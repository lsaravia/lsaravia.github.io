Este es un curso de posgrado sobre metapoblaciones utilizando lenguaje R, basicamente es una introducción a la modelación de metapoblaciones utilizando el modelo cĺásico de Levins y agregando estocacidad como error de proceso, error de observación, usando tiempo continuo con procesos de Markov y sistemas de particulas interactivas para hacerlo espacialmente explícito.
